# DOI

DOIs are made into full doi links, if they are not already. 

```
doi  = "https://doi.org/" + doi;
```

# Author names 

The [author name parser](https://github.com/liliana-sanfilippo/author-name-parser) is used and names are formatted depending on the citation style.
